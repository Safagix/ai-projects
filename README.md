# AI Projects

Public-safe collection of AI experiments and assistant prototypes from Digital Lab.

## Contents

- `Eira`, `Eira V2`, `Eira_App`, `Eira_Chat`, `Eira_Nexos_Client`, `Eira_Nexus`: Eira assistant experiments, UI prototypes, local memory concepts, and app interfaces.
- `Heira`: Telegram/local-first AI assistant prototype with tool routing and local model support.
- `local-rag-md`: local Markdown RAG system with SQLite/FTS, Ollama-oriented configuration, and web UI.
- `human-typer`: human-like typing automation research project.
- `magic-pointer-mcp`: pointer/control MCP prototype with screen and action modules.
- `EiraLearn`: learning assistant/product prototype.

## Public Safety

This repository is a cleaned source export. Runtime data and machine-specific files were intentionally excluded: `.env` files, local databases, vector indexes, logs, virtual environments, build folders, dependency folders, packaged binaries, embedded local Node runtimes, and generated voice outputs.

API keys that appeared in historical files were replaced with placeholders or environment variable reads. Local paths were normalized to `%DIGITAL_LAB%` or `%USERPROFILE%` where useful.

See `SECURITY_PUBLISH_NOTES.md` for the publishing audit notes.
