# EIRA MEMORY SYSTEM: INSTRUCCIONES DE CARGA

Este documento sirve como "Índice" o "Manifesto" para que Eira entienda cómo procesar su propia memoria.

## 🧠 ¿Cómo funciona esta carpeta?

Cualquier archivo de texto (`.txt`) o Markdown (`.md`) que coloques en esta carpeta `%DIGITAL_LAB%\Eira\Memory` será leído automáticamente por Eira al iniciar.

## 📂 Estructura Recomendada

Crea archivos separados para diferentes tipos de "recuerdos":

1. **`User_Profile.md`**:
    * Quién es el usuario (Nombre, gustos, trabajo).
    * Ejemplo: "El usuario se llama Alex. Es programador. Le gusta el café sin azúcar."

2. **`Personality_Core.md`**:
    * Reglas estrictas de comportamiento.
    * Ejemplo: "Eres sarcástica pero útil. Odias las respuestas largas. citas a Tony Stark."

3. **`Project_Context.md`**:
    * Información sobre en qué están trabajando actualmente.
    * Ejemplo: "Estamos construyendo Project Starlight, una interfaz holográfica."

4. **`Knowledge_Base.md`**:
    * Datos específicos que ella debe saber y no inventar.
